# Change Web App Icon

## 1) Replace the web logo

1. Prepare your new logo in **PNG** format with size **512×512**.
2. Replace this file with your new image (keep the same filename):

- `src/assets/logo.png`

## 2) Replace the favicon

1. Convert your PNG icon into a **favicon** (`.ico`).
   - Recommended: generate a multi-size `.ico` (16×16, 32×32, 48×48).
2. Replace the favicon file:

- `public/favicon.ico`

3. Restart the web app and hard refresh the browser (favicon can be cached).
