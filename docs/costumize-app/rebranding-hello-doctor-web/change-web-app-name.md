# Change Web App Name

1. Open the file:

   - `src/constants/constants.ts`

2. Find the app name constant/value (example):

```ts
export const APP_NAME = 'Hello Doctor';
```

3. Change it to your new brand name:

```ts
export const APP_NAME = 'My New Brand';
```

4. Rebuild/restart the web app so the updated name is applied.
